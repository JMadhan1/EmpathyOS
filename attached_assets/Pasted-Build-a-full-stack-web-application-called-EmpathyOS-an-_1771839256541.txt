Build a full-stack web application called "EmpathyOS" — an AI-powered micro-coach for difficult workplace conversations, grounded in the "Love as a Strategy" framework.

=== TECH STACK ===
- Frontend: Pure HTML, CSS, JavaScript (no frameworks)
- Backend: Python Flask
- LLM: OpenAI API (GPT-4o) via Flask backend
- No database needed for demo (use session/localStorage)

=== CORE CONCEPT ===
EmpathyOS helps employees and managers prepare for and reflect on difficult workplace conversations using AI coaching grounded in empathy, respect, courageous honesty, and shared purpose (Love as a Strategy pillars).

The app has ONE main conversation flow with 4 steps:
Step 1 – CLARIFY: User describes the situation, goal, and fears
Step 2 – DRAFT: User writes what they want to say; AI returns a coached version
Step 3 – ANTICIPATE: AI simulates 3 likely emotional reactions from the other person + loving responses
Step 4 – REFLECT: After the conversation, user logs what happened; AI gives micro-learning insight

=== BACKEND (app.py + prompts.py) ===

Flask routes:
- GET / → serve index.html
- POST /api/clarify → takes {scenario, situation, goal, worry, other_feeling} → returns {reframed_intent, blind_spot, emotional_context}
- POST /api/draft → takes {draft_text, context from step 1} → returns {coached_version, coaching_notes: [{emoji, note}]}
- POST /api/anticipate → takes {full context} → returns {reactions: [{type, emoji, they_might, respond_with}]}
- POST /api/reflect → takes {what_happened, surprise, proud, different, full context} → returns {what_you_did_with_love, micro_habit, love_reminder}

Use Flask session to carry context across steps (store step 1 and 2 data in session).

prompts.py — System prompt for ALL routes:
"""
You are EmpathyOS, an AI micro-coach for difficult workplace conversations.
You are grounded in the "Love as a Strategy" framework, which treats love as a serious business strategy built on:
- Empathy & Understanding
- Respect & Psychological Safety
- Courageous Honesty
- Shared Purpose & Trust
- Vulnerability & Openness
- Accountability with Care

Your role is to:
1. Help people prepare for difficult conversations with love, courage, and clarity
2. Never replace the human conversation — only augment the human's ability to show up well
3. Always prioritize human dignity, mutual respect, and relationship health
4. Be warm, direct, and practical — not preachy or generic
5. Give specific, actionable coaching — not vague affirmations

Tone: Warm, wise, direct. Like a trusted coach who believes in both love AND accountability.
Format: Always return valid JSON matching the exact schema requested.
"""

Each route has its own user prompt (in prompts.py) that passes the user's inputs and requests a specific JSON schema back.

Error handling:
- If OpenAI call fails, return a graceful fallback JSON with a warm error message
- All API responses return JSON with proper HTTP status codes
