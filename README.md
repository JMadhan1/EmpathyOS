# EmpathyOS: Scaling Love as a Strategy 💛

**The Human Layer of the Organizational Tech Stack.** 

EmpathyOS is an AI-powered culture platform built for the **Love X AI: Bangalore Innovation Challenge**. It transforms the "6 Pillars of Love" into measurable business metrics, moving beyond individual coaching to true organizational intelligence.

## 🚀 Key Innovation Features

### 1. Organizational Intelligence Dashboard
- **Quantifying Love**: A real-time Radar Chart measuring the 6 Pillars (Empathy, Safety, Honesty, Purpose, Vulnerability, Care) across the company.
- **Psychological Safety Trends**: 30-day tracking of organizational sentiment to correlate "Loving Leadership" with team velocity.
- **Executive Insights**: AI-generated manager action items based on real conversation dynamics.

### 2. Culture Repair Engine (Team Rituals)
- **Beyond the Person**: When a conflict occurs, EmpathyOS suggests a **Team Ritual** (e.g., *Blocker Breakfast* or *Worry Window*) to fix the root cause of the environment, not just the individual.

### 3. Multi-Channel Export
- **One-Click Scaling**: Instantly export AI-coached reframes to **Slack** (with formatting/emojis) or **Email** (professional templates) to maintain culture across all digital touchpoints.

### 4. Human-First AI Coaching
- **Clarify Intent**: Untangle defensive emotions before they turn into toxic communication.
- **Anticipate Reactions**: AI mapping of potential employee defensiveness to prepare leaders for "Fear-based" pushback.

---

## 🛠️ Technology Stack

- **Frontend**: React, Vite, Framer Motion (Premium Animations), Tailwind CSS.
- **Backend**: Node.js, Express.
- **AI**: Google Gemini API (Reframing, Ritual Generation, Narrative Analysis).
- **Database**: SQLite with Drizzle ORM (Persistent Culture Logs).
- **Charts**: Recharts (Radar & Time-series visualization).

## 📦 Getting Started

1.  **Clone the Repo**:
    ```bash
    git clone https://github.com/JMadhan1/EmpathyOS.git
    cd EmpathyOS
    ```
2.  **Environment Variables**:
    Create a `.env` file:
    ```env
    GEMINI_API_KEY=your_api_key_here
    ```
3.  **Install & Run**:
    ```bash
    npm install
    npm run dev
    ```
